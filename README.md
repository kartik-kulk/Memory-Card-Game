# Memory-Card-Game
A small python memory card game made using Turtle library. 
Features: 
 - Can play with 8, 10 or 12 cards
 - Tracks leaderboards, showing best 8 scores so far
 - A quit button with credits
 - The ability to load your own deck onto the game. Guide to upload in ```design.txt```

Detailed game design description in ```design.txt``` file. To run the game: 
 - Ensure all game files ```config.txt``` ```pictures.txt``` and ```leaderboard.txt``` and all sprite gif images are in the same directory. Missing files will show respective error images on the game screen, and the program will not be interactable.
 - Run the ```memory_game.py``` file from the game files through python IDLE and enjoy
